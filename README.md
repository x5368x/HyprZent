## ✨ Sobre HyprZent <span id="about"></span>

**HyprZent** es la configuración definitiva para Hyprland:  
un entorno visual, moderno y zen con glass, neón y máxima personalización para tu setup Linux.

- Visual glassy/neón/gradient
- Modular, rápido y fácil de instalar
- Experiencia zen: limpia, minimalista y espectacular
- Soporte multi-distro y multiplataforma

---

## 🚀 Instalación <span id="install"></span>

> **Recomendado en Arch y derivados.**

```bash
pacman -S --needed git base-devel
git clone --depth 1 https://github.com/HyDE-Project/HyDE ~/HyDE
cd ~/HyDE/Scripts
./install.sh
```
- Añade apps extra a `Scripts/pkg_user.lst` y pásalo como parámetro:  
  `./install.sh pkg_user.lst`
- Configuraciones personalizadas se respaldan en `~/.config/cfg_backups`.

## 🧩 Características <span id="features"></span>

- Glass, gradient & neon en todo el entorno
- Waybar, Rofi, swww, wlogout, dunst, playerctl, grimblast, wl-clipboard, cava, pywal, kitty, foot, tmux, fastfetch, zenfetch, etc.
- Scripts inteligentes: wallpapers, energía, QR sharing, layouts, autotheme, notifications, IA.
- Automatización y modularidad total
- Exporta y comparte por QR/URL y visual generator
- Integración IA (auto-sugerencias, asistentes beta)
- Interfaz móvil y web (próximamente)

## 🔗 Integraciones <span id="integrations"></span>

- **Hyprland** (rolling release) y todos sus plugins
- Compatible con Arch, Fedora, Debian, Ubuntu, NixOS, Void, Alpine, openSUSE, Artix, CachyOS, Gentoo…
- Dotfiles extra, plantillas, scripts portables
